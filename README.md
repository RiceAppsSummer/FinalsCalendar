# INTRODUCTION

FinalSchedule is an app for Rice students to visualize their final schedules better.

### Frontend Framework
- [jQuery] FullCalendar
- [BootStrap]
- [HTML/CSS]

### Backend
- [sqlLite]
- [Flask]
- [Python 2.7]

### Flask Extensions
- [SQLAlchemy]
- [pdfMiner]

### Usage
- Install Flask
- Setup venv 
- python setup.py in command line
- python runserver.py
